## CaintGram for Android

[CaintGram](https://github.com/Nesonaragirl/CaintGram) is a modern, highly customizable Telegram client for Android built on the official Telegram source code.

The project focuses on delivering a flexible and extensible messaging experience through a powerful plugin system, advanced customization, and a native Android user experience while remaining fully compatible with Telegram's ecosystem.

> **Warning**
>
> CaintGram is currently a **Work In Progress (WIP)**.
>
> Development builds are **not intended for daily use** and may contain severe bugs, incomplete features, crashes, and other stability issues.
>
> A closed testing phase will begin before the first public release.

## Features

Current development includes:

* Plugin Framework
* Performance Improvements
* Privacy-Focused Design
* Continuous Integration

Additional functionality will be introduced throughout development.
